# BetterScreenShot

A Windows screenshot app built with C# and WPF.

## Features test

- Full screen capture
- Select area capture
- Per-monitor DPI-aware selection overlay
- Automatic save to `Pictures/BetterScreenShot`
- Bottom-right preview toast with `Open` and `Delete`
- Screenshot viewer with `Save As`

## Run locally

1. Open `BetterScreenShot.sln` in Visual Studio.
2. Press `F5` to run the application.
